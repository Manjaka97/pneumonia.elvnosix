ResNet using PyTorch

Dataset

The training data consists of 5216 X-ray scans of pediatric patients from one to five years old from Guangzhou Women and Children's Medical Center, Guangzhou. The model was then tested on a test set consisting of 624 images it has not seen before.

Model

This ConvNet is a ResNet18 initially pretrained with ImageNet. After trying to use it just as a feature extractor and obtaining unsatisfying results, it was entirely re-trained. You can see the complete implementation of the model on github here.

Results

Note that for an imbalanced dataset in the medical field, Recall and F1 score are more significant.
Accuracy 90.2%
Precision 87.8%
Recall 97.9%
F1 Score 92.6%
